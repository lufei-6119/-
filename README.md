# Minecraft Server Manager

一个基于 Node.js 的 Minecraft 服务器管理面板

## ✨ 功能

* 本机信息监控（CPU / 内存 / 磁盘）
* 防火墙状态查看
* Minecraft 服务器控制

  * 启动 / 停止
  * 实时日志（彩色）
  * 命令输入
* 文件上传
* 一键打开服务器目录

## 🚀 使用方法

### 1. 安装 Node.js

下载并安装 Node.js

### 2. 启动项目

双击运行：

```
start.bat
```

或命令行：

```
node server.js
```

### 3. 打开浏览器

访问：

```
http://你的IP:8080
```

## 📁 目录说明

* `server.js` 后端逻辑
* `index.html` 前端界面
* `MinecraftServer/` 存放服务器文件（自动创建）

## ⚠️ 注意

* 不要将 MinecraftServer 上传到 GitHub
* 确保已安装 Java 才能运行服务器

## 🧠 技术栈

* Node.js
* 原生 HTML + JS
* Windows 系统命令
